Struct RowEncodingOptions 

pub struct RowEncodingOptions(/* private fields */);
Expand description

Options for the Polars Row Encoding.

The row encoding provides a method to combine several columns into one binary column which has the same sort-order as the original columns.test

By default, the row encoding provides the ascending, nulls first sort-order of the columns.
