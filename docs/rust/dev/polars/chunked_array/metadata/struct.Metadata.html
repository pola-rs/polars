pub fn merge(&self, other: Metadata<T>) -> MetadataMerge<T>

Merge the maximum information from both Metadatas into one Metadata.

It returns

  • MetadataMerge::Keep if the self already contains all the information
  • MetadataMerge::New(md) if we have learned new information
  • MetadataMerge::Conflict if the two structures contain conflicting metadata
