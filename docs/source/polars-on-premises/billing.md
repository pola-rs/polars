# Billing

Polars On-Prem Self Serve includes a generous free tier with no credit card required. Beyond the
free tier, usage is billed at a pay-as-you-go rate. Volume discounts are available.
[Contact us](mailto:support@pola.rs) to discuss.

## Self Serve

|                      |                            |
| -------------------- | -------------------------- |
| Free tier            | 10,000 CPU hours per month |
| Beyond free tier     | $0.05 per vCPU/hour        |
| Credit card required | No                         |

### Limits

The following limits apply to Self Serve deployments:

|                           |       |
| ------------------------- | ----- |
| Concurrent clusters       | 1     |
| Maximum cores per cluster | 1,024 |
| Maximum nodes per cluster | 64    |

## Enterprise

Enterprise is priced based on your deployment. It removes resource limitations and supports fully
air-gapped environments with no outbound connectivity required.
