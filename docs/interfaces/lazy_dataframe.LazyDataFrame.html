fetch

  • fetch(numRows?: number): DataFrame
  • fetch(numRows: number, opts: LazyOptions): DataFrame

  • Fetch is like a collect operation, but it overwrites the number of rows read by every scan

    Note that the fetch does not guarantee the final number of rows in the DataFrame. Filter, join operations and a lower number of rows available in the scanned file influence the final number of rows.

filter

  • filter(predicate: string | Expr): LazyDataFrame

  • Filter the rows in the DataFrame based on a predicate expression.

    example
    >>> lf = pl.DataFrame({
    >>>   "foo": [1, 2, 3],
    >>>   "bar": [6, 7, 8],
    >>>   "ham": ['a', 'b', 'c']
    >>> }).lazy()
    >>> // Filter on one condition
    >>> lf.filter(pl.col("foo").lt(3)).collect()
    shape: (2, 3)
    ┌─────┬─────┬─────┐
    │ foo ┆ bar ┆ ham │
    │ --- ┆ --- ┆ --- │
    │ i64 ┆ i64 ┆ str │
    ╞═════╪═════╪═════╡
    │ 1   ┆ 6   ┆ a   │
    ├╌╌╌╌╌┼╌╌╌╌╌┼╌╌╌╌╌┤
    │ 2   ┆ 7   ┆ b   │
    └─────┴─────┴─────┘

    Parameters

    • predicate: string | Expr

      Expression that evaluates to a boolean Series.

    Returns LazyDataFrame
