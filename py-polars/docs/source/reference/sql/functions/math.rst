.. _abs:

ABS
---
Returns the absolute value of the input column.

**Example:**

.. code-block:: python

    df = pl.DataFrame({"a": [-1.0, 0.0, 1.0, -2.0]})
    df.sql("""
      SELECT a, ABS(a) AS abs_a FROM self
    """)
    # shape: (4, 2)
    # ┌──────┬───────┐
    # │ a    ┆ abs_a │
    # │ ---  ┆ ---   │
    # │ f64  ┆ f64   │
    # ╞══════╪═══════╡
    # │ -1.0 ┆ 1.0   │
    # │ 0.0  ┆ 0.0   │
    # │ 1.0  ┆ 1.0   │
    # │ -2.0 ┆ 2.0   │
    # └──────┴───────┘
